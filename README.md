### 🚀 Featured Project
**vibhavmacos.vercel.app** — A fully functional **macOS Simulator** in the browser with 13+ apps:

- ✅ Interactive Finder (create, copy, delete, folders)
- ✅ AI-Powered Terminal (30+ commands)
- ✅ Spotify, YouTube, Photos, VS Code, Maps
- ✅ 3D Futuristic Mode (Avaturn + Mixamo animated avatar + Three.js room)
- ✅ Game, Resume, Contact & more

Built with **Next.js, Tailwind, Firebase, Google Gemini & React Three Fiber**
